# Colour swap profiles for infrared photography

This repository contains forty profiles for Photoshop's Camera Raw and Lightroom/Lightroom Mobile. They should work with any camera model, provided that the base models (created with DNG Profile Editor) named "Infrared Temp -100" and "Infrared Temp -50" have been imported beforehand.
